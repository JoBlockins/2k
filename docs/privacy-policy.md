# Privacy Policy — 2K Erg Training App

**Last updated:** February 15, 2026

## Overview

This is a personal training application built for individual use. It is not a commercial product and is not available to the public.

## Data Collected

This app accesses the following data from your WHOOP account via the WHOOP Developer API:

- Recovery scores, HRV, and resting heart rate
- Sleep duration, stages, and quality metrics
- Daily strain and workout data
- Basic profile and body measurements

## How Data Is Used

All data is used solely for personal training analysis and performance tracking. Data is stored locally on the developer's machine and is never shared with third parties.

## Data Storage

Data is stored in a local SQLite database. No data is transmitted to external servers beyond the AI coaching API (Anthropic Claude) for training analysis.
